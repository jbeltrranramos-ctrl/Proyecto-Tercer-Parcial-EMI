# 🤖 Robot Autónomo Seguidor de Línea

Proyecto académico de robótica que consiste en el diseño, construcción y programación de un robot móvil autónomo capaz de seguir una línea mediante sensores infrarrojos. El proyecto integra conocimientos de ecuaciones diferenciales, estadística, diseño 3D, microcontroladores, física y programación, aplicados a un sistema real.

---

## 📌 Descripción general

El robot utiliza sensores infrarrojos para detectar una trayectoria y un sistema de control PID para corregir su movimiento en tiempo real. El chasis es diseñado en 3D y el control se implementa sobre un microcontrolador, permitiendo analizar el desempeño del sistema mediante datos estadísticos como tiempo, error y estabilidad.

---

## 🎯 Objetivos

### Objetivo general
Desarrollar un robot seguidor de línea autónomo que aplique conceptos teóricos y prácticos de diversas áreas de la ingeniería.

### Objetivos específicos
- Implementar un control PID para el seguimiento de línea  
- Diseñar un chasis funcional mediante modelado 3D  
- Analizar el rendimiento del robot usando estadística  
- Aplicar principios físicos del movimiento  
- Programar el sistema en un microcontrolador  

---

## 🧠 Integración de materias

- **Ecuaciones diferenciales:** modelado del error y control PID  
- **Estadística:** análisis de tiempos, errores y repetibilidad  
- **Diseño 3D:** diseño del chasis y soporte de componentes  
- **Microcontroladores:** lectura de sensores, PWM y control de motores  
- **Física:** cinemática, fricción, torque y estabilidad  
- **Programación:** lógica de control, funciones y máquina de estados  

---

## 🛠️ Componentes principales

- Microcontrolador (Arduino / ESP32 u otro)
- Sensores infrarrojos
- Motores DC con caja reductora
- Driver de motores
- Chasis diseñado en 3D
- Fuente de alimentación

---

## ⚙️ Funcionamiento básico

1. Los sensores detectan la posición de la línea  
2. Se calcula el error respecto al centro  
3. El controlador PID corrige la velocidad de los motores  
4. El robot ajusta su trayectoria en tiempo real  

---

## 📊 Análisis y pruebas

Durante las pruebas se recopilan datos como:
- Tiempo de recorrido  
- Error de seguimiento  
- Estabilidad en curvas  

Estos datos se analizan estadísticamente para evaluar el desempeño del sistema y realizar mejoras.
